A Data-Based Web Framework

You write SQL. EcoSystem does the rest.

EcoSystem makes it quick and enjoyable to develop sites and apps using the power of PostgreSQL to specify your data and logic requirements.

Key Features

  • Easy to get started. Just write SQL!
  • Written in Golang - fast and easy to deploy.
  • Organise, share and reuse data and logic with "bundles".
  • Ready to use off-the-shelf - no need to learn Go, but...
  • ...fully extensible with simple Go packages: easy to create a custom build.
  • Standard build includes core API, JWT auth and image resizing.
  • Roadmap for packages includes: admin dashboard, HTML web server, Hugo integration for static sites, auto-generating API docs and more...

Complete technical overview

Built For Developers

EcoSystem is a development platform centred around a JSON API server written in Go. When developing on EcoSystem, you code data and logic in PostgreSQL and let EcoSystem take care of the boilerplate. EcoSystem is open source (and always will be), lightweight, has a shallow learning curve and deliberately seeks to avoid complexity at all levels.

EcoSystem is an open-source platform for building custom, web-connected, data-driven software; it doesn't make any assumptions about your business requirements, allowing you to build only and exactly what you need - and quickly.
