EcoSystem is for Developers

EcoSystem is a toolset consisting of a web and API server written in Go and a versatile admin panel in Polymer. As an EcoSystem ERP developer, you write PostgreSQL, supply the web templates, and configure the admin panel with JSON. EcoSystem is open source (and always will be), lightweight, has a shallow learning curve and deliberately seeks to avoid complexity at all levels.
